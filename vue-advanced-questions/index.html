created() {
    this.compoData = this.store.getters('someGetter')
}

и затем при добавлении данных в хранилище необходимо за ними проследить

watch: {
    'store.getters' (newValue) {
          this.compoData = newValue
    }
}

Answer

необходимо сделать ваш ватчер immediate true

'store.getters': {
    heandler (newValue) {

  1. Существует ситуация когда нужно при инициализации компонента к примеру подписать на событие DOM а перед тем когда компонент будет унечтожен отписаться от события Это возмоэно сделать через доболнительную переменную коммпонента в которую можно сохранить подписку а затем делать отписку в нужный момент.

Answer

Более компактное решение:

created() {
   const escapeHandler = e => {
      if (e.key === 'Escape' && this.show) {
        this.cancel()
      }
    }
    document.addEventListener('keydown', escapeHandler)
    this.$once('hook:destroyed', () => {
      document.removeEventListener('keydown', escapeHandler)
    })
}
